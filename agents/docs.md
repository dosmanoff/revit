/// </param>

/// </returns>
/// <exception cref="ArgumentNullException">

/// </exception>
/// <exception cref="Autodesk.Revit.Exceptions.InvalidOperationException">

/// </exception>
/// <example>
/// <code>
/// using var t = new Transaction(doc, "Create walls");
/// t.Start();
/// var service = new WallCreationService(doc);

/// t.Commit();
/// </code>
/// </example>
public IList<Wall> CreateWalls(Document document, IList<Curve> contour, string levelName)

## Документирование Revit API-специфики

При документировании компонентов, работающих с Revit API, **всегда указывай**:

### Контекст выполнения
```csharp
/// <remarks>
/// ⚠️ Должен вызываться из главного потока Revit.
/// Для вызова из фонового потока используйте <see cref="IExternalEvent"/>.
/// </remarks>
```

### Требования транзакции
```csharp
/// <remarks>
/// ✅ Требует открытую транзакцию (<see cref="Transaction"/>).
/// Метод не открывает транзакцию самостоятельно.
/// </remarks>
```

### Версионные различия
```csharp
/// <remarks>
/// Revit 2024: использует устаревший метод <c>get_Parameter()</c>.
/// Revit 2025: использует <c>GetParameter()</c>.
/// Класс обеспечивает единый интерфейс для обеих версий.
/// </remarks>
```

### Производительность
```csharp
/// <remarks>
/// ⚡ Использует быстрый фильтр <c>ElementClassFilter</c>.
/// Не используй для коллекций > 10 000 элементов — рассмотри пакетную обработку.
/// </remarks>
```

### Шаблон страницы команды
```markdown
# Команда: Создать стены

## Что делает
Создаёт стены по замкнутому контуру на выбранном уровне.

## Где найти
Ribbon → Вкладка "BIM Tools" → Панель "Стены" → Кнопка "Создать по контуру"

## Параметры
| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| Тип стены | Да | Выбор из загруженных типов |
| Уровень | Да | Базовый уровень привязки |
| Высота | Нет | По умолчанию: высота уровня |

## Шаги
1. Откройте план этажа
2. Нарисуйте замкнутый контур линиями модели
3. Выберите линии
4. Нажмите "Создать по контуру"
5. Выберите тип стены и уровень в диалоге
6. Нажмите OK

## Ограничения
- Контур должен быть замкнутым
- Минимум 3 сегмента
- Поддерживаются только прямые линии (не дуги)
```
