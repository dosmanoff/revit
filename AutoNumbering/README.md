# AutoNumbering — плагин для Revit 2025

Плагин автоматически присваивает последовательные номера выбранным элементам модели и записывает их в параметр (по умолчанию **Mark**).

---

## Установка

1. Собери проект или возьми готовую сборку из `bin/Release/net8.0-windows/`.
2. Скопируй два файла в папку аддинов Revit:
   ```
   C:\ProgramData\Autodesk\Revit\Addins\2025\
       AutoNumbering.dll
       AutoNumbering.addin
   ```
3. Запусти Revit. При первом запуске подтверди загрузку плагина.

---

## Запуск

Вкладка **Smart Tools** → панель **Numbering** → кнопка **Auto Numbering**.

---

## Работа с диалогом

### 1. Выбор элементов

- Если перед запуском команды в модели были выделены элементы — они подставятся автоматически.
- Если ничего не выделено — Revit предложит выбрать элементы вручную (зажми Ctrl для множественного выбора, затем нажми **Finish** или зелёную галку).
- Кнопка **Re-select Elements…** — закрывает диалог и возвращает в модель для нового выбора, сохраняя текущие настройки.

Поддерживаемые категории: стены, плиты, балки, колонны, арматурные стержни и любые другие элементы физической модели.

### 2. Настройка нумерации

| Поле | Описание | Пример |
|------|----------|--------|
| **Prefix** | Текст перед номером | `W-` |
| **Start** | Начальное число | `1` |
| **Step** | Шаг нумерации | `1` |
| **Min digits** | Минимальное количество цифр (с ведущими нулями) | `3` → `001` |
| **Suffix** | Текст после номера | `-A` |

Строка **Preview** показывает, как будут выглядеть первые два номера.

### 3. Параметры

- **Write to** — параметр, в который записывается номер. По умолчанию `Mark`. Можно выбрать любой другой текстовый параметр из выпадающего списка или ввести вручную.
- **Sort by** — параметр, по которому сортируется список перед нумерацией (например, `Level`, `Comments`). Выбери `(none)` чтобы не сортировать.
- **Ascending / Descending** — направление сортировки.

### 4. Таблица элементов

Таблица показывает все выбранные элементы. Можно:

- **Снять галочку** напротив элемента — он будет пропущен, нумерация продолжится без него.
- **Кликнуть по заголовку колонки** — пересортировать список. Порядок строк = порядок нумерации.

Колонки:
- **✓** — включить/исключить элемент
- **Category** — категория Revit
- **Family / Type** — тип семейства
- **Sort Value** — значение параметра, выбранного в «Sort by»
- **Current Value** — текущее значение параметра «Write to»
- **New Value** — предлагаемый номер (зелёным)

### 5. Применение

Нажми **OK** — плагин запишет номера в модель за одну операцию (отменяется через Ctrl+Z).

По завершении появится итоговое сообщение: сколько элементов пронумеровано и были ли ошибки.

---

## Примеры использования

**Нумерация стен с префиксом:**
Prefix = `СТ-`, Start = `1`, Step = `1`, Min digits = `2` → `СТ-01`, `СТ-02`, …

**Нумерация с шагом 10:**
Prefix = ``, Start = `10`, Step = `10` → `10`, `20`, `30`, …

**Нумерация по уровням:**
Sort by = `Level`, Ascending → элементы пронумеруются снизу вверх.
